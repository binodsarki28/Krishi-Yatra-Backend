KrishiYatra - Backend (Spring Boot)

Project Overview
KrishiYatra is a farmer-to-consumer delivery system designed to eliminate middlemen and streamline agricultural supply chains using a stock-based delivery platform.


Core Features

JWT Authentication and Spring Security (RBAC)
Specialized dashboards for Farmers, Buyers, Riders, and Admins
Firebase Cloud Messaging (FCM) for push notifications
Cloudinary integration for product image management
Live order tracking with Leaflet Maps integration
Automated delivery fee calculations and marketplace engine



Technology Stack

Java, Spring Boot, MySQL
Firebase, Cloudinary, JWT, Flyway



Setup Instructions

Clone the repository.
Update database and cloud credentials in application.properties.
Run the application
